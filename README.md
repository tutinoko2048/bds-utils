### bds-updater

A CLI tool for updating Bedrock Dedicated Server.

**Features:**
- Check for the latest BDS version
- Download and install the latest version
- Switch between versions

**Installation:**

Via npm:
```bash
npm install -g bds-updater
```

Or download the executable directly:
https://github.com/tutinoko2048/bds-utils/releases/latest

**Usage:**

Via installed package:
```bash
bds-updater
```

Or run directly with using bun:
```bash
bunx bds-updater
```
